Welcome to the Editor Menu!


On this page, you will find the following:
1. A button that returns scripts relevant to the currently-logged in editor.
You may also approve a script from the viewing.
2. A button that displays any messages from an author.
3. A button that displays any messages from another editor.
A) For the editor to type out a request for further information.
4. A button that allows the editor to request more information from an author.
A) For the senior editor, this gives the script the final approval.
5. A button that will allow you to reject a script. You must provide a reason for doing so.
6. If a story is marked High Priority, only those stories can be interacted with.
7. A logout button.
